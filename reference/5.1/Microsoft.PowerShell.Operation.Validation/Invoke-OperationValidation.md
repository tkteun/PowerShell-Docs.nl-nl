# Invoke-OperationValidation

## SAMENVATTING

Hiermee worden tests voor het valideren van de bewerkings raamwerk aangeroepen.

## SYNTAXIS

### FileAndTest (standaard)

```
Invoke-OperationValidation [-TestInfo <PSObject[]>] [-IncludePesterOutput] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### Pad

```
Invoke-OperationValidation [-testFilePath <String[]>] [-IncludePesterOutput] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### UseGetOperationTest

```
Invoke-OperationValidation [-ModuleName <String[]>] [-TestType <String[]>] [-IncludePesterOutput] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## BESCHRIJVING

Met de cmdlet **invoke-OperationValidation** worden bewerkings validatie raamwerk tests aangeroepen voor een opgegeven module.
