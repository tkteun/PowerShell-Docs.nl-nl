# Publish-DscConfiguration

## SAMENVATTING
Publiceert een DSC-configuratie naar een set computers.

## SYNTAXIS

### ComputerNameSet (standaard)

```
Publish-DscConfiguration [-Path] <String> [-Force] [[-ComputerName] <String[]>] [-Credential <PSCredential>]
 [-ThrottleLimit <Int32>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### CimSessionSet

```
Publish-DscConfiguration [-Path] <String> [-Force] [-ThrottleLimit <Int32>] -CimSession <CimSession[]>
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHRIJVING
De cmdlet **Publish-DscConfiguration** publiceert een configuratie document van de desired state Configuration (DSC) van Windows Power shell op een set computers.
Met deze cmdlet wordt de configuratie niet toegepast.
Configuraties worden toegepast met behulp van de cmdlet Start-DscConfiguration als deze wordt gebruikt met de para meter *UseExisting* of wanneer de DSC-engine de consistentie cyclus uitvoert.
De DSC-engine wordt ook wel de lokale Configuration Manager (LCM) genoemd.

Deze cmdlet is vooral handig wanneer er fragmenten van meerdere configuratie documenten worden geleverd.
Wanneer er meerdere fragmenten voor configuratie documenten worden geleverd, overschrijven ze de oudere configuratie document fragmenten.

## VOORBEELDEN

### Voor beeld 1: een configuratie naar een externe computer publiceren

```
PS C:\> Publish-DscConfiguration -Path '$home\WebServer' -ComputerName "ContosoWebServer" -Credential (get-credential Contoso\webadministrator)
```

Met deze opdracht wordt een configuratie naar een externe computer gepubliceerd.
De gebruiker die de cmdlet uitvoert, moet beheerder zijn op de externe computer.
