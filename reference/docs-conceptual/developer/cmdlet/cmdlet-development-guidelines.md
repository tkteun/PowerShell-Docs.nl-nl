# <a name="cmdlet-development-guidelines"></a>Richtlijnen voor de ontwikkeling van cmdlets

De onderwerpen in deze sectie bieden ontwikkel richtlijnen die u kunt gebruiken om goed opgemaakte cmdlets te maken. Door gebruik te maken van de algemene functionaliteit van de Windows Power shell-runtime en door deze richt lijnen te volgen, kunt u robuuste cmdlets met minimale inspanning ontwikkelen en de gebruiker een consistente ervaring bieden. Daarnaast vermindert u de test last omdat de algemene functionaliteit niet opnieuw moet worden getest.
