# <a name="hidetableheaders-element-format"></a>Het element HideTableHeaders (opmaak)

Hiermee geeft u op dat de kopteksten van de tabel niet worden weer gegeven.

ViewDefinitions element (indeling) element weer geven (indeling) TableControl element (Format) HideTableHeaders element voor TableControl (indeling)

## <a name="syntax"></a>Syntaxis

```vb
<HideTableHeaders/>
```

## <a name="attributes-and-elements"></a>Kenmerken en elementen

In de volgende secties worden de kenmerken, onderliggende elementen en het bovenliggende element van het element `HideTableHeaders` beschreven.

### <a name="attributes"></a>Kenmerken

Geen.

### <a name="child-elements"></a>Onderliggende elementen

Geen.

### <a name="parent-elements"></a>Bovenliggende elementen

|Element|Beschrijving|
|-------------|-----------------|
|[TableControl-element (indeling)](./tablecontrol-element-format.md)|Definieert een tabel indeling voor een weer gave.|

## <a name="text-value"></a>Tekst waarde

Geef `true` op om de kopteksten van de tabel te verbergen.
