# Clear-ItemProperty

## SAMENVATTING
Hiermee wordt de waarde van een eigenschap gewist, maar wordt de eigenschap niet verwijderd.

## SYNTAXIS

### Pad (standaard)

```
Clear-ItemProperty [-Path] <String[]> [-Name] <String> [-PassThru] [-Force] [-Filter <String>]
 [-Include <String[]>] [-Exclude <String[]>] [-Credential <PSCredential>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### LiteralPath

```
Clear-ItemProperty -LiteralPath <String[]> [-Name] <String> [-PassThru] [-Force] [-Filter <String>]
 [-Include <String[]>] [-Exclude <String[]>] [-Credential <PSCredential>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## BESCHRIJVING

Met de cmdlet wordt de `Clear-ItemProperty` waarde van een eigenschap gewist, maar de eigenschap wordt niet verwijderd.
U kunt deze cmdlet gebruiken om de gegevens uit een register waarde te verwijderen.

## VOORBEELDEN

### Voor beeld 1: de waarde van de register sleutel wissen

Met deze opdracht worden de gegevens in de register waarde ' Options ' van de subsleutel ' Mijntoep ' gewist `HKEY_LOCAL_MACHINE\Software\MyCompany` .

```powershell
Clear-ItemProperty -Path "HKLM:\Software\MyCompany\MyApp" -Name "Options"
```

## OPMERKINGEN

- U kunt gebruiken `Clear-ItemProperty` om de gegevens in register waarden te verwijderen zonder de waarde te verwijderen.
  Als het gegevens type van de waarde binary of DWORD is, wordt met het wissen van de gegevens de waarde ingesteld op nul.
  Anders is de waarde leeg.
- De `Clear-ItemProperty` cmdlet is ontworpen om te werken met de gegevens die door elke provider worden weer gegeven. Als u een lijst wilt weer geven van de providers die beschikbaar zijn in uw sessie, typt u `Get-PSProvider` . Zie [about_Providers](../Microsoft.PowerShell.Core/About/about_Providers.md)voor meer informatie.
