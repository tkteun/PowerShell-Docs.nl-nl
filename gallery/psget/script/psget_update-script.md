# <a name="update-script"></a>Update-Script

Updatescript cmdlet kunt u in-place bijwerken van de scriptbestanden die zijn geïnstalleerd met de cmdlet Install-Script.

## <a name="description"></a>Beschrijving

De cmdlet updatescript werkt het opgegeven script uit de opslagplaats waaruit deze eerder was geïnstalleerd.

## <a name="cmdlet-syntax"></a>De syntaxis van cmdlet

```powershell
Get-Command -Name Update-Script -Module PowerShellGet -Syntax
```
## <a name="cmdlet-online-help-reference"></a>Verwijzing naar het online help van cmdlet

[Update-Script](http://go.microsoft.com/fwlink/?LinkId=619787)
